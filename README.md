# Inventory Manager

A desktop inventory management app built with Python and CustomTkinter.

## Features
- Add, restock, and sell items
- Live search
- Tracks quantity and price
- Auto saves to JSON file
- Modern dark UI

## Built With
- Python
- CustomTkinter

## How to Run
1. Install dependencies: `pip install customtkinter`
2. Run: `python Project1.py`
